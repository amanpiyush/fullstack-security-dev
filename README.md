# 🚀 Full Stack Developer with a Focus on Web Security

> Building applications. Breaking them. Securing them.

---

## 🧠 About This Repository

This repository documents my journey of becoming a **Full Stack Developer with a strong focus on Web Application Security**.

Instead of just learning development, I follow a practical approach:

* Build real-world applications
* Identify vulnerabilities in them
* Fix and secure the implementations

This helps me understand both **how systems are built** and **how they can be exploited**.

---

## ⚔️ Core Philosophy

> Build → Break → Secure

* **Build** → Develop functional web applications
* **Break** → Test for vulnerabilities (IDOR, XSS, Auth flaws)
* **Secure** → Implement proper fixes and best practices
